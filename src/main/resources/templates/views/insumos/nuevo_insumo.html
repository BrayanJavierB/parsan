Creación de Insumo

El ID del insumo debe ser mayor que 0
El nombre debe tener entre 3 y 50 caracteres
La descripción debe tener entre 5 y 200 caracteres
La clase debe tener entre 3 y 50 caracteres
El NIT del proveedor debe tener entre 3 y 20 caracteres
El nombre del proveedor debe tener entre 3 y 50 caracteres
